# Fraud Detection with Explainable AI (SHAP)

This project develops a machine learning-based fraud detection system capable of identifying fraudulent transactions while providing transparent and interpretable predictions through SHAP (SHapley Additive Explanations).

The workflow includes data preprocessing, exploratory data analysis, outlier treatment, model training using Random Forest, performance evaluation, and explainability analysis. SHAP visualizations are used to understand feature contributions and explain individual predictions, making the model more trustworthy and suitable for real-world financial applications.

## Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Outlier detection and handling
- Random Forest classification model
- Fraud prediction interface using ipywidgets
- Model evaluation using confusion matrix, ROC curve, and classification metrics
- Explainable AI with SHAP feature importance and prediction explanations

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- SHAP
- Matplotlib
- Seaborn
- Google Colab

## Objective
To build an accurate fraud detection model while ensuring transparency and interpretability through Explainable AI techniques.
